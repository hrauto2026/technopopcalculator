Dear Customer, Thank you for using RS-2026A. Please read these instructions carefully.

01. Mode Selection

Toggle between Calculation and Music modes using the light-yellow button. Note that the display marker will track your current mode.

02. Universal Synthesis

This unit maintains square-wave audio feedback in both modes. Calculations are audible.

03. Memory & Reset

In Music mode, use [Del Rec]. In Calculator mode, this becomes the [AC] function for clearing registers.

04. Return Key

The [RETURN] key performs a standard backspace on the current active input digit or musical note.
